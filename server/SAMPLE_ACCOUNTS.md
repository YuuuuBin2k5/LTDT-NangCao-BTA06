## 🗑️ Xóa Dữ Liệu Mẫu

Để xóa tất cả dữ liệu mẫu:

```sql
-- Xóa reviews
DELETE FROM reviews WHERE user_id IN (SELECT id FROM users WHERE email LIKE '%@sample.com');

-- Xóa friendships
DELETE FROM friendships WHERE user_id_1 IN (SELECT id FROM users WHERE email LIKE '%@sample.com');

-- Xóa current_locations
DELETE FROM current_locations WHERE user_id IN (SELECT id FROM users WHERE email LIKE '%@sample.com');

-- Xóa users
DELETE FROM users WHERE email LIKE '%@sample.com';

-- Xóa places (nếu cần)
DELETE FROM places WHERE id > 0;
```
